Cold Staking Setup

In order to set up cold staking you need to run a hot staking node and have a cold wallet containing the funds you want to stake.

Hot staking node
You can only stake funds in your cold wallet when this node is online and actively staking. A hot staking node is required prior to setting up your cold wallet. If you're using a third party service to run the hot staking node, you can skip this step and get the hot staking address from the service.
Cold wallet
If you want to set up a cold wallet on this device, click create cold staking wallet.
